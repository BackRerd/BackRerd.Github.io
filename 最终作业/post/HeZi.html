HTML 中的盒模型教程

欢迎来到 HTML 盒模型的基础教程。在这个教程中，我们将了解 HTML 元素的盒模型，以及如何使用 CSS 控制盒模型的外观。

什么是盒模型？

在 HTML 中，每个元素都被视为一个矩形框，这个矩形框包含了内容区、内边距、边框和外边距。这被称为盒模型。

盒模型的组成部分

HTML 元素的盒模型由以下四个主要部分组成：

  • 内容区（Content）：显示元素的实际内容，如文本、图像等。
  • 内边距（Padding）：围绕在内容区的空白区域，用于分隔内容区和边框。
  • 边框（Border）：围绕在内边距外的边框，用于定义元素的边界。
  • 外边距（Margin）：是元素边框外的区域，用于分隔元素与其他元素的距离。

盒模型示例

下面是一个简单的 HTML 元素的盒模型示例：

            
<div style="width: 200px; padding: 20px; border: 2px solid #333; margin: 30px;">
    这是一个示例文本。
</div>

在这个例子中，<div> 元素的宽度是 200 像素，内边距为 20 像素，边框为 2 像素的实线，外边距为 30 像素。

使用 CSS 控制盒模型

您可以使用 CSS 样式表来控制元素的盒模型外观。以下是一些基本的 CSS 样式：

            
.box {
    width: 200px;
    padding: 20px;
    border: 2px solid #333;
    margin: 30px;
}

通过将这个样式应用到 HTML 元素，您可以控制元素的宽度、内边距、边框和外边距。
