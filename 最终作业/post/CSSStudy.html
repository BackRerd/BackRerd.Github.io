什么是 CSS？

CSS（层叠样式表）是一种用于描述文档展示样式的样式表语言。它使您能够将文档的样式与其结构分离，从而实现更好的可维护性和可重用性。

基本选择器

选择器是一种用于选择要应用样式的 HTML 元素的模式。以下是一些基本选择器：

            
/* 通过元素名称选择 */
p {
    color: blue;
}

/* 通过类选择 */
.my-class {
    font-size: 16px;
}

/* 通过 ID 选择 */
#my-id {
    background-color: #f0f0f0;
}

盒模型

在 CSS 中，每个元素被视为一个矩形框，由内容、内边距、边框和外边距组成。这就是所谓的盒模型。

            
.box {
    width: 200px;
    height: 100px;
    padding: 10px;
    border: 2px solid #333;
    margin: 20px;
}

布局和定位

CSS 提供了各种布局和定位属性，使您能够精确控制元素的位置和大小。

            
/* 居中元素 */
.center {
    width: 50%;
    margin: 0 auto;
}

/* 相对定位 */
.relative {
    position: relative;
    top: 10px;
    left: 20px;
}

/* 绝对定位 */
.absolute {
    position: absolute;
    top: 50px;
    right: 20px;
}
