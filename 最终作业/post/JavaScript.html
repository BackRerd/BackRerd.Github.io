JavaScript 简介

JavaScript 是一种脚本语言，用于为网页添加交互性和动态效果。它通常与 HTML 和 CSS 一起使用，形成现代 web 开发的基石。

基本语法

JavaScript 的基本语法包括变量、数据类型、运算符、控制流语句等。

            
// 变量和数据类型
let name = "John";
const age = 25;
var isStudent = true;

// 运算符
let result = 10 + 5;
result *= 2;

// 控制流语句
if (age >= 18) {
    console.log("成年人");
} else {
    console.log("未成年人");
}

DOM 操作

DOM（文档对象模型）是 JavaScript 操作网页内容的接口。通过 DOM，可以动态修改网页的结构和样式。

            
// 获取元素
let element = document.getElementById("myElement");

// 修改内容
element.innerHTML = "新内容";

// 修改样式
element.style.color = "red";

事件处理

JavaScript 允许您对网页上的事件（例如点击、鼠标悬停）进行响应。

            
// 添加点击事件监听器
let button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("按钮被点击了！");
});
