Python 简介

Python 是一种易学、强大且高级的编程语言。它以清晰简洁的语法和丰富的标准库而著称。

安装 Python

首先，确保您已经安装了 Python。您可以从官方网站（https://www.python.org/）下载并安装最新版本。

你的第一个 Python 程序

让我们编写一个简单的“Hello, World!”程序来入门：

            
print("Hello, World!")

将此代码保存为名为 hello_world.py 的文件。然后在终端中运行：

            
python hello_world.py

这将在控制台上显示“Hello, World!”。

变量和数据类型

在 Python 中，您可以使用变量存储数据，而无需显式声明变量的类型。常见的数据类型包括整数、浮点数和字符串。

            
age = 25
price = 19.99
message = "Hello, Python!"

控制流语句

Python 支持条件语句（if、else）和循环语句（for、while）。以下是一个简单的例子：

            
x = 10

if x > 5:
    print("x 大于 5")
else:
    print("x 不大于 5")

函数

在 Python 中，您可以定义函数来组织代码。以下是一个简单的例子：

            
def add(a, b):
    return a + b

result = add(5, 3)
print("结果：", result)
